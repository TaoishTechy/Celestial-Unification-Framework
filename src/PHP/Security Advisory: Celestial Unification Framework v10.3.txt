Security Advisory: Celestial Unification Framework v10.3
Version 10.3 "Integrity Edition" is a mandatory security update that resolves multiple critical vulnerabilities and hardening gaps present in all prior versions.

Summary
This release focuses on defense-in-depth, hardening the framework against supply-chain attacks, insecure configuration, data corruption, and information disclosure. All users should upgrade immediately.
